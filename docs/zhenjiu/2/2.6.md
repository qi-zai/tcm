# 十二经纳之井荣俞经合
十二正经，奇经八脉

## 十二正经
### ☁️ 六阴经
- 手太阴肺经
- 手厥阴心包经
- 手少阴心经
- 足太阴脾经
- 足厥阴肝经
- 足少阴肾经

### 🌞 六阳经
- 手太阳小肠经
- 手阳明大肠经
- 手少阳三焦经
- 足太阳膀胱经
- 足阳明胃经
- 足少阳胆经

## 奇经八脉
任脉、督脉、冲脉、带脉、阳蹻，阴蹻、阳维、阴维

## 井荣俞经合

```
   阳经              阴经
    金   -   井   -   木
    水   -   荣   -   火
    木   -   俞   -   土
    火   -   经   -   金
    土   -   合   -   水
```

> 阴经和阳经每一条经上都有络，叫络穴；阳经有原穴。

## 病在脏取井穴
病在脏取井穴，通通在手指头或脚趾头末梢。

## 病在脏取荣穴 - 颜色
遇有颜色出现，取荣穴；如脸发黑，就是肾脏病。

下针相对较痛

## 病在脏取俞穴 - 时间
也可叫 shu 穴，按照时间发病的情况，可取，如晚上12点起来，病发于胆经上，取胆经的俞穴。

## 下针注意
井荣俞经合都是手不过肘，足部过膝
